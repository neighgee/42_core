# 42 Core Projects

This repository contains the projects I have completed so far during my 42 journey.  
Each project helped me strengthen my understanding of C programming, problem-solving, algorithms, memory management, and system-level concepts.

## Completed Projects
- **libft** — A custom C library with reimplemented standard functions and additional utilities.
- **ft_printf** — A recreation of the `printf` function with support for formatted output.
- **get_next_line** — A function that reads and returns a file line by line.
- **so_long** — A small 2D game project focused on graphics, event handling, and map validation.
- **minitalk** — A client-server communication program using UNIX signals.
- **push_swap** — A sorting algorithm project focused on efficiency and stack operations.
- **minishell** — A simple shell that handles commands, pipes, redirections, and environment variables.
- **philosophers** — A multithreading project based on the dining philosophers problem, exploring synchronization and concurrency.

These projects reflect my progress in low-level programming and my experience with the 42 curriculum.
